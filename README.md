# Mart.ps QA Testing Project

This repository contains my contribution to a university QA assignment for the Software Quality Assurance (QA) course.

## Project Overview
The assignment focused on performing manual testing for the Palestinian e-commerce website Mart.ps.

The work included:
- Writing user stories
- Designing test cases
- Executing manual tests
- Reporting software bugs
- Organizing QA workflow using Trello

## My Contribution
This was a group assignment, and this repository includes only the parts I personally worked on.

My tasks included:
- Creating user stories
- Writing test cases
- Executing test cases
- Reporting and documenting bugs
- Capturing testing screenshots
